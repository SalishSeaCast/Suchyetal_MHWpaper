# Suchyetal_MHWpaper analysis code

[![DOI](https://zenodo.org/badge/1094459397.svg)](https://doi.org/10.5281/zenodo.17584028)

This repository contains interactive Jupyter notebooks and Python scripts that will allow for reproducability of the results and figures presented in:

>Suchy, K. D., Allen, S. E., A. R. Sastri, & K. Young.  Modelling the impacts of marine heatwaves on plankton in the Salish Sea.
